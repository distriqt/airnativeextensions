UPDATE_EULERConstant 
public static const UPDATE_EULER:String = devicemotion:update:euler

Dispatched when the extension has an update about the current Euler angles of the device. You must have registered with euler angles specified as the format option.

The values array will contain the azimuth, pitch and roll in that order.

values = [ azimuth, pitch, roll ] 
         var azimuth:Number     = event.values[0];
         var pitch:Number     = event.values[1];
         var roll:Number     = event.values[2];

UPDATE_QUATERNIONConstant 
public static const UPDATE_QUATERNION:String = devicemotion:update:quaternion

Dispatched when the extension has an update about the current quaternion of the device. You must have registered with quaternion specified as the format option.

The values array contains the details of the current quaternion in the following order:

values = [ w, x, y, z ] 
         var q:Quaternion = new Quaternion( event.values[1], event.values[2], event.values[3], event.values[0] );
