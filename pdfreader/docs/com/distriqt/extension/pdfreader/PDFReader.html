This class represents the PDF Reader extension.

The PDF Reader allows you to display an interactive PDF reader/viewer to your user.

The extension provides some simple functionality to display a PDF in the reader and then several events around opening, and closing the reader.

iOS

On iOS this extension uses the VFR Reader library that can be found here:

  • Documentation: http://www.vfr.org/
  • Source Code: https://github.com/vfr/Reader

setToolbarOptions()method 
public function setToolbarOptions(showEmail:Boolean, showBookmarks:Boolean, showThumbnails:Boolean):Boolean

Sets which of the controls will be visible when a PDF is displayed to the user. These are the controls along the top bar in the viewer.

If you do not call this function then all the controls are shown.

Parameters

showEmail:Boolean — If true the "email" button will be available to allow the user to email the PDF
 
showBookmarks:Boolean — If true the "bookmarks" button will be shown to allow the user to bookmark pages in the document
 
showThumbnails:Boolean — If true the "thumbnails" button will be shown to allow the user to switch to a thumbnail view of the document

Returns
Boolean — true if successful and false if the function is not supported

Examples

This simple example shows the basic functionalty of how to display a PDF in the reader. The PDF file is a file named "Reader.pdf" and packaged in the application directory with the application. 

     //
     // Initialise the extension
     PDFReader.init( DEV_KEY );
     
     //
     // Show a PDF
     var path:String = File.applicationDirectory.nativePath + File.separator + "Reader.pdf";
     PDFReader.service.showPDF( path );
