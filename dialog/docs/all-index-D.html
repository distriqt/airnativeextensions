dismissDialog(id:int) — method, class com.distriqt.extension.dialog.Dialog
Dismiss a dialog that you have previously displayed. The id of the dialog must be provided so that if you have stacked dialogs the correct dialog will be dismissed. This does not dismiss progress dialogs only dialogs created with the #showAlertDialog or #showMultipleChoiceDialog functions. If called at the appropriate time, you will receive a DialogEvent.DIALOG_CANCELLED event after this dialog is dismissed. iOS: Under iOS the dialogs take a little time to initialise so make sure you leave at least 1-2 seconds between creating the dialog and dismissing it.
