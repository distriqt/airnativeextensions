autoScaleproperty 
public var autoScale:Boolean = false

The autoScale feature allows you to specify position and size based on the devices dip which will automatically scale the popover based on the device display resolution.

For example, the following would display a popover of width 200 pixels and height 200 pixels on an iPhone 3GS (non-retina) positioned on a box of 20x20 pixels at 150,100. However on an iPhone 5 (retina) this would appear as a box of 400 pixels positioned on a box of 40x40 at 300,200. 

         options.autoScale = true;
         options.position = new Rectangle( 150, 100, 20, 20 );
         options.size = new Rectangle( 200, 200 );

The default is false, allowing the position and size to align with the AIR pixel coordinate system.

multiSelectproperty 
public var multiSelect:Boolean = false

If true then the user can select multiple items in the list, otherwise they can only select one.

positionproperty 
public var position:Rectangle

This rectangle specifies an area on the screen that the popover will be "attached" to. The native popover will best fit in order to associate the content with this area. Generally this should the be size and location of a button that was pressed by the user to initiate this popover.
