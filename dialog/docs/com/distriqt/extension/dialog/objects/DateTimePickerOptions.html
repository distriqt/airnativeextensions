Packagecom.distriqt.extension.dialog.objects
Classpublic class DateTimePickerOptions
InheritanceDateTimePickerOptions Object

This class controls the display options for the date and time dialogs.

It allows you to control the title, labels and other display options along with initial values displayed in the dialog.

See also

com.distriqt.extension.dialog.Dialog.showDateTimePickerDialog()

MODE_DATEConstant

MODE_DATE_AND_TIMEConstant 
public static const MODE_DATE_AND_TIME:String = mode:dateAndTime

In this mode the dialog will show the ability to select a date (year, month, day) and a time (hour, minute).

Note: This is only currently supported on iOS

MODE_TIMEConstant 
public static const MODE_TIME:String = mode:time

In this mode the dialog will only show the ability to select a time, i.e. an hour and minute.
