mimeTypeproperty 
mimeType:String

A mimetype to use for the attachment. Not required but makes the email attachments appear better in some mail applications.

This should be the fully qualified mime type, eg:

  • image/jpeg
  • image/png
  • text/plain

nativePathproperty 
nativePath:String

The native path on the device to the file to use as the attachment. The best way to pass this is to use an instance of a File object and pass the File.nativePath variable.
