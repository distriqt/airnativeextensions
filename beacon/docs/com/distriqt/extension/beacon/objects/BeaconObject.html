Packagecom.distriqt.extension.beacon.objects
Classpublic class BeaconObject
InheritanceBeaconObject Object

This class represents a beacon that was encountered during region monitoring. You do not create instances of this class directly. The extension reports encountered beacons. You can use the information in a beacon object to identify which beacon was encountered.

The identity of a beacon is defined by its UUID, major, and minor properties. These values are coded into the beacon itself.
