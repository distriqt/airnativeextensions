pollingEnabledproperty 
pollingEnabled:Boolean  [write-only]

This allows you to enabled or disable the polling for network interface changes.

This is enabled automatically when you first access the extension however if you've since disabled the polling this function allows you to restart it.

If you stop the polling you will not receive the CHANGE events when the network interfaces are updated.
