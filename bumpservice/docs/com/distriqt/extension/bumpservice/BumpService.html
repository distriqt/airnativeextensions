Classpublic class BumpService
InheritanceBumpService com.distriqt.extension.bumpservice.base.ExtensionBaseImpl

This class represents an Native Extension interface wrapper for the Bump API.

This Native Extension requires the additional use of the Bump API with an API key. See http://bu.mp/company/api for information.

Notes and Limitations

This ANE currently only supports sending String data between devices. Other data types may be included in the future if requested.

Developers should manage the connection of the Bump service manually for an application. By default, if connected, the service will continue to run if your application is moved to the background state. This functionality may be unwanted and should be handled appropriately by calling BumpService.service.disconnect() when your application enters a state in which this is not required.
