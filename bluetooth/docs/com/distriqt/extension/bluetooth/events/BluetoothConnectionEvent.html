Packagecom.distriqt.extension.bluetooth.events
Classpublic class BluetoothConnectionEvent
InheritanceBluetoothConnectionEvent flash.events.Event

This event describes connection events related to the listening and connecting functionality of the bluetooth extension.

This event will contain the majority of the information about a connection over its lifetime

Generally event event will contain at least the uuid of the connection that this event is related to. If a device was involved then the device will contain the information about the remote device involved in the connection.
