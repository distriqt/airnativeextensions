Packagecom.distriqt.extension.facebookapi.events
Classpublic class FacebookAPIEvent
InheritanceFacebookAPIEvent flash.events.Event

This event class is used for most notifications from the FacebookAPI ANE.

Note that you can also register callback functions for most ANE calls, however some actions will not support callbacks and you must rely on listening for these event types.

The event will generally include a data and error property, which you can inspect for information about the event.

Due to the different response data returned from Facebook, these properties may not always be parsed and returned properly objects or strings. In these cases, you can usually inspect the rawData string for additional information about the response.
