Classpublic class FacebookOpenGraphActionEvent
InheritanceFacebookOpenGraphActionEvent flash.events.Event

This event class is used for information about Facebook open graph story IDs.

When a user is directed to your application from tapping an Open Graph story link on Facebook, this information is available in your app at startup.

When the application is opened, if any open graph story data is found, the ACTION_IDS_FOUND event will be dispatched with a list of IDs representing the open graph story. This is likely to only be a single ID but will be contained within an array.

If you wish to load additional data about the open graph story, you can do so by simply loading a graph API request with the method "GET" and passing the story ID as the URL.
