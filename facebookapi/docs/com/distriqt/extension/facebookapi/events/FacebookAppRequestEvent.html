Classpublic class FacebookAppRequestEvent
InheritanceFacebookAppRequestEvent flash.events.Event

This event class is used for information about Facebook app requests.

When a user is directed to your application from tapping an app request link on Facebook, this information is available in your app at startup.

When the application is opened, if any app request data is found, the APP_REQUESTS_FOUND event will be dispatched. You can then use the app request IDs passed in the requestIds parameter to load more information about the request by using the FacebookAPI.loadAppRequestInfo method.

Once loaded, the APP_REQUEST_DATA_LOADED event will be dispatched containing a list of FacebookAppRequestInfo objects.

NOTE: Once you have loaded and processed the app request info, you should then delete the requests from Facebook. You can do this using the FacebookAPI.deleteAppRequests method.
