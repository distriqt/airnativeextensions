Classpublic class FacebookOpenGraphStoryParams
InheritanceFacebookOpenGraphStoryParams Object

This class contains available data to be used for publishing open graph stories to Facebook.

In order to use this functionality, you must configure the Open Graph settings in the Facebook app dashboard for your application.

More information is available at https://developers.facebook.com/docs/opengraph

explicitlySharedproperty 
public var explicitlyShared:Boolean = false

This option specifies that this story should be explicitly published to the user's timeline, rather than just in the activity log.

You should ensure your usage complies with the Facebook guidelines for explicit sharing. See the guidelines at https://developers.facebook.com/docs/opengraph/using-actions#explicitsharing

NOTE: You must specifically enable the "Explicitly Shared" capability in the open graph settings for your action in the Facebook app dashboard.

isUserGeneratedImageproperty 
public var isUserGeneratedImage:Boolean = false

Set this to true when the attached image is a photo from the camera or similar. This makes the image appear larger like a photo post, rather than a thumbnail for the OG story.

To use this setting, your image must be large enough for Facebook's requirements. Small images will not be accepted here and will cause an error.

You must specifically enable the "User Generated Photos" capability in the open graph settings for your action in the Facebook app dashboard.

See also

https://developers.facebook.com/docs/opengraph/using-actions.photos

placeIdproperty 
public var placeId:String

A Facebook Place ID for this story action to be tagged at.

NOTE: You must specifically enable the "Place" capability in the open graph settings for your action in the Facebook app dashboard.

WARNING: We recommend only using this parameter when you are publishing an open graph action through the openGraphStoryWithAPI method under Android. Tests have shown the Facebook library does not always handle this parameter when using the OG dialogs.
