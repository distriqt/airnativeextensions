This class represents the properties available when sending a graph request

The Graph API Explorer is a very useful tool for testing functionality, and can be found at https://developers.facebook.com/tools/explorer

See also

https://developers.facebook.com/docs/graph-api
https://developers.facebook.com/tools/explorer/
