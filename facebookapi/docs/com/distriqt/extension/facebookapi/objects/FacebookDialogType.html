Classpublic class FacebookDialogType
InheritanceFacebookDialogType Object

This class specifies the available dialog types to be used for Facebook share and request dialogs.

Please note that not all dialog types are supported for different actions, or on both iOS and Android.

Generally the ANE will use the preferred dialog type you set, but fall back to another type if required.

NATIVEConstant
public static const NATIVE:String = fbDialogType:native

This type will use the native Facebook application for dialogs where available. This type will only work if the user has the native Facebook application installed on their device.

Supported for:

  • Login and permission request dialogs
  • Share dialogs
  • Open Graph story dialogs

OS_INTEGRATEDConstant 
public static const OS_INTEGRATED:String = fbDialogType:osIntegrated

This type will use the native OS-integrated UI for Facebook which was introduced in iOS6.

This type of dialog requires that the user has connected their Facebook account in the iOS settings.

Supported for:

  • Login and permission request dialogs on iOS6+
  • iOS Facebook Share Sheet dialog (with limited parameters)

Using this dialog type has the following limitations:

  • You cannot tag friends with this dialog type
  • You cannot specify a "to" user with this dialog
  • Posts with this dialog are attributed to iOS, rather than your application name (i.e. shared "via iOS")

WEBConstant 
public static const WEB:String = fbDialogType:web

This dialog type is the generic fallback, which opens a modal web view above the application.

Web dialogs are generally available for all types, with the exception of open graph story posts.

Supported for:

  • Login and permission request dialogs
  • Share dialogs
  • App request dialogs
