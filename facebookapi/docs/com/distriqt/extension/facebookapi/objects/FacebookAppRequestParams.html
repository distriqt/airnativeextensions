redirect_urlproperty 
public var redirect_url:String

The URL to redirect to after a person clicks a button on the dialog. Required when using URL redirection.

This should generally not be required for mobile applications.

suggestedFriendsproperty 
public var suggestedFriends:Array

A list of Facebook user IDs to be shown in the friend selection dialog.

If not empty, only the given friends will be shown for selection in the friend picker.

titleproperty 
public var title:String

The title for the app request dialog. Maximum length is 50 characters.

toproperty 
public var to:Array

An array containing a list of user's Facebook IDs to send this request to.

If empty, the user will be able to select friends from a list in the dialog.
