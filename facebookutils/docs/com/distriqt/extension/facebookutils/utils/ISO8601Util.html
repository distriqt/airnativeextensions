public function parseDateTimeString(val:String):Date

converts an ISO-8601 date + time (UTC) string of format "2009-02-21T09:00:00Z" to an AS3 Date Object. The zulu designation (Z) at the end of the string indicates the time is UTC (Coordinated Universal Time). Even if the zulu designation is missing UTC will be assumed.
