Packagecom.distriqt.extension.facebookutils
Classpublic class FacebookUtils
InheritanceFacebookUtils com.distriqt.extension.facebookutils.base.ExtensionBase

This class is a simple interface to implementing Facebook SSO login.

Ensure you add the URL scheme information to your application XML file, as shown with the included example application.

Replace "YOUR_FACEBOOK_APP_ID" with the application ID of your Facebook app. Your application type should be set to Mobile/Desktop in the advanced application settings on Facebook. Make sure to keep the "fb" prefix before the ID.
