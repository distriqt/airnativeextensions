Camera Native Extension Library

This extension creates access to native features of the Camera hardware giving you finer control over the camera and allowing you to create more complex imaging applications.

Key Features
  • Preview frames (ByteArray of pixel data for easy display to Bitmap)
  • Take a photo and simulataneously save to "camera roll" and retrieve as BitmapData
  • Control focus mode, exposure, white balance
  • Control focus point and exposure point
  • List available cameras and switch between cameras

Supported on the following platforms:

  • Android: All unless stated otherwise in documentation
  • iOS: All unless stated otherwise in documentation
Operation Suggestion

Alot of these functions require access to the camera so for example to check whether a mode is support on a camera we have to open the camera, request access and then query the camera. This can be a slow operation and if you put together a sequence of these calls you may find your application stuttering.

To get around this we suggest that you make sure your first operation is to select and initialise a device. Once the device is initialised you can query the features and modes of this device and the extension will use the initialised camera instance instead of having to open up a new one.

For example the two code samples below the second will run faster. 

     var devices:Array = Camera.instance.getAvailableDevices();
     var device:CaptureDevice = devices[0];
     
     var options:CameraParameters = new CameraParameters();
     options.deviceId = device.id;
     
     Camera.instance.isFlashSupported( device.id );
     Camera.instance.isFocusSupported( CameraParameters.FOCUS_MODE_AUTO, device.id );
     
     Camera.instance.initialise( options );
     
     var devices:Array = Camera.instance.getAvailableDevices();
     var device:CaptureDevice = devices[0];
     
     var options:CameraParameters = new CameraParameters();
     options.deviceId = device.id;
     
     Camera.instance.initialise( options );
     
     Camera.instance.isFlashSupported();
     Camera.instance.isFocusSupported( CameraParameters.FOCUS_MODE_AUTO );
iOS

Under iOS the user will be requested access to the camera when you attempt to access it.

Android

Under Android you'll need to add the camera permission to your manifest additions, and you should also add the "uses-feature" if your application relies on the camera for core functionality (This will limit your application to being installed only on devices with a camera). You can safely leave the uses-feature option off if you plan to conditionally support the camera. 

	 <uses-permission android:name="android.permission.CAMERA"/>
	 <uses-feature android:name="android.hardware.camera" />

If you wish to save images/videos to the device then you'll need to make sure you've requested permission to write to the storage by adding the following to the manifest additions: 

	 <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

getFrameBuffer()method 
public function getFrameBuffer(destination:ByteArray):int

Copies the latest frame captured by the camera to the supplied byte array.

The returned data will be in little endian format which we set on the ByteArray instance. Additionally we reset the position to the start of the ByteArray so it will be ready for use immediately after a call to this function.

Parameters

destination:ByteArray — A valid ByteArray instance to write the frame data into

Returns
int — The index of the received frame data, -1 if there was an error, eg the ByteArray was invalid

See also

height

Retrieving the data for a video frame 
         var videoDataByteArray:ByteArray = new ByteArray();
         
         Camera.instance.getFrameBuffer( videoDataByteArray );

             new Rectangle( 0, 0, Camera.instance.width, Camera.instance.height ), 
             videoDataByteArray 
         );

Examples
The following code initialises and turns on the flash 
         try
         {
             Camera.init( "your_dev_key" );
             Camera.instance.initialise();
             Camera.instance.setFlashMode( CameraParameters.FLASH_MODE_TORCH );
         }
         catch
         {
             trace( "ERROR: "+e.message );
         }
The following code captures bitmap data from the camera 
     
     private var _lastFrameProcessed    : Number = -1;
     private var _videoData            : ByteArray = null;
     private var _bitmapData            : BitmapData;
     private var _bitmap                : Bitmap;
     
     ...
     
         try
         {
             Camera.init( "your_dev_key" );
     
             _videoData = new ByteArray();
             _bitmapData = new BitmapData( 320, 240, true );
             _bitmap = new Bitmap( _bitmapData );
     
             Camera.instance.initialise();
             Camera.instance.addEventListener( CameraEvent.VIDEO_FRAME, camera_videoFrameHandler, false, 0, true );
         }
         catch
         {
             trace( "ERROR: "+e.message );
         }
     
     ...
     
     private function camera_videoFrameHandler( event:CameraEvent ):void
     {
         var frame:Number = Camera.instance.receivedFrames;
         if (frame != _lastFrameProcessed)
         {
             Camera.instance.getFrameBuffer( _videoData );
             var rect:Rectangle = new Rectangle( 0, 0, Camera.instance.width, Camera.instance.height );
             if (_bitmapData.width != Camera.instance.width || _bitmapData.height != Camera.instance.height)
             {
                 _bitmapData = new BitmapData( Camera.instance.width, Camera.instance.height, true );
                 _bitmap.bitmapData.dispose();
                 _bitmap.bitmapData = _bitmapData;
             }
             
             try
             {
                 _bitmapData.setPixels( rect, _videoData );
             }
             catch (e:Error)
             {
                 trace( e.message );
             }
             
             _videoData.clear();
             _lastFrameProcessed = frame;
         }
     }
The following example shows how to capture an image using the built in camera and retrieve the image as BitmapData while simultaneously saving the image to the user's camera roll / gallery. 
     
     private var _bitmapData            : BitmapData;
     private var _bitmap                : Bitmap;     
     
     ...
      
         if (Camera.isSupported)
         {
             Camera.init( "your_dev_key" );
             var options:CameraParameters = new CameraParameters();
             options.enableFrameBuffer = false;
             options.presetMode = CameraParameters.PRESET_PHOTO;
     
             try
             {
                 Camera.instance.initialise( options );
                 Camera.instance.addEventListener( CameraDataEvent.CAPTURED_IMAGE, camera_capturedImageHandler, false, 0, true );
                 Camera.instance.addEventListener( CameraEvent.IMAGE_SAVE_COMPLETE, camera_imageSaveCompleteHandler, false, 0, true );
                 
                 Camera.instance.captureImage( true );
             }
              catch
             {
             }
         }
     
     ...
     
     private function camera_capturedImageHandler( event:CameraDataEvent ):void
     {
         trace( "capture complete" );
         // event data contains the bitmap data
         _bitmapData.draw( event.data );
     }
     
     private function camera_imageSaveCompleteHandler( event:CameraEvent ):void
     {
         // event.data contains the asset url
         trace( "image save complete: " +event.data );
     }
