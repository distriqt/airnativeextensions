Property Detail
developerPayloadproperty
public var developerPayload:String

This is a ‘developer payload’ string that you can use to send supplemental information about an order (it can be an empty string).

Typically, this is used to pass in a string token that uniquely identifies this purchase request.

errorproperty 
public var error:String

A description of the error, empty if none occurred.

errorCodeproperty 
public var errorCode:String

An error code, will be empty if none occurred.

originalMessageproperty 
public var originalMessage:String

This will contain the actual contents of the message received by the in app billing extension. Under iOS this will always be empty, but under Google Play it will contain the JSON formatted purchase message.

originalPurchaseproperty 
public var originalPurchase:Purchase = null

In the case of a restored transaction, this will contain details about the original purchase transaction.

productIdproperty

quantityproperty 
public var quantity:int = 1

The quantity associated with this transaction. Under Google Play this should always be 1.
