This class represents a product you have created in your applications store.

It contains all the product information retrieved about the product, such as id, title, description etc

priceStringproperty 
public var priceString:String

The product price represented as a string ready for user display. This will contain the currency sign of the price.

Wherever possible you should use this string to display the amount of the product.

sourceproperty 
public var source:String

This will be the original source string of the product matching what the store has sent to the device. Use this if you want to do any specific processing on the product information that you can't perform with the above.

Generally you won't need to use this
