_pixelScaleproperty 
protected var _pixelScale:Number = 1

The pixel scale to be used when drawing this image

This is 1 by default. For iOS retina displays, this should be set to 2. You should create two versions of marker icons to allow retina display support, and use NativeMaps.service.isRetinaDisplay() to decide which icon to use when creating markers.
