Constant Detail
MAGNETIC_FIELD_AVAILABLEConstant
public static const MAGNETIC_FIELD_AVAILABLE:String = magnetic:field:available

Dispatched when the magnetic field updates are available and have been initialised. After this event you can expect to receive magnetic field updated events.

The field values are not valid with this event and will be set to 0.

MAGNETIC_FIELD_UNAVAILABLEConstant 
public static const MAGNETIC_FIELD_UNAVAILABLE:String = magnetic:field:unavailable

Dispatched when the magnetic field updates are unavailable. This could be due to the device not containing a compass or if the extension was initialised incorrectly.

The field values are not valid with this event and will be set to 0.

MAGNETIC_FIELD_UPDATEDConstant 
public static const MAGNETIC_FIELD_UPDATED:String = magnetic:field:updated

Dispatched when the magnetic field changes on the device. This event will contain values in the #fieldX #fieldY #fieldZ parameters specifying the current magnetic values.
